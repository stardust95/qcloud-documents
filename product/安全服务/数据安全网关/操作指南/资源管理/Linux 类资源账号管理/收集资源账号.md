## 操作场景
在您需要将 Linux 资源上大量的系统帐号添加到到数据安全网关系统时，可以通过数据安全网关系统的“抽取”功能收集账号。该指南指导您如何收集 Linux 资源账号。


## 前提条件
需事先配置资源的管理员账号，具体操作请查看 [配置管理员账号](https://cloud.tencent.com/document/product/1025/32220) 文档。


## 操作步骤
>!如果数据安全网关上不存在与 Linux 资源相同的账号，使用“抽取”功能时，则数据安全网关系统直接添加账号。如果两者存在相同账号，抽取时 Linux 帐号则会覆盖数据安全网关上的账号。 

1. 使用管理员账号登录数据安全网关系统。
2. 在数据安全网关系统单击【资源管理】>【Linux】，即可进入 Linux 资源管理页面。
3. 找到您要获取账号的资源，在账号一列下单击，即可进入资源账号配置页面：
![1](https://main.qcloudimg.com/raw/b04669730acd62048622192f6211c8d3.png)
4. 在资源账号配置页面，单击【抽取】，即可从远程资源收集账号到数据安全网关系统。
![2](https://main.qcloudimg.com/raw/0cdc74403c0a10f97156ea83c69bd028.png)
